# Nota-Venta-SpringBoot
Este proyecto esta hecho de manera educativa con fines de aprender como se utiliza el framework 
de Spring Boot y su utilidad en proyectos tipo empresariales, si bien es un CRUD sencillo me ha servido 
mucho para afianzar conocimientos.
Todo este proyecto a sido creado de manera autodidacta y teniendo una pagina en Notion para el seguimiento del 
proyecto, desde su planificacion, implementacion y posterior documentacion.
